Jak udělat discord bota v javascriptu ?

Krok 1.

Udělej si účet se na replit.com.

Krok 2.

Najdi tlačítko New repl a na to klikni.

Vyber Discord.js starter.

Krok 3.

Smaž všechny soubory kromě index.js.

V souboru index.js všechny kódy smaž.

Krok 4.

Napiš tam tenhle kód

const Discord = require("discord.js");
const client = new Discord.Client();

client.on("ready", () => {
    console.log(`Link na bota: https://discord.com/oauth2/authorize?client_id=${client.user.id}&permissions=8&scope=bot`);
})

client.on("message", message => {

    // Příkaz 1 
    /* Místo "ahoj" napište příkaz, například pokud tam 
        napíšete "!test", a pak napíšete na váš discord server "!test" tak to odešle 
        tu zprávu kterou jste napsali v tom .send()*/
    
    if(message.content === "ahoj") {
        return message.channel.send("Ahoooj" + message.author.tag);
    }

    // Příkaz 2
    if(message.content === "!server") {
        return message.channel.send("Discord server: https://miftik.tk/discord");
    }

    // Příkaz 3
    if(message.content === "!otazka") {
        return message.channel.send(`Jak se máš ${message.author.tag} ?`);
    }

    // Embeds návod
    // To \n tam rozdělí řádky

    if(message.content === "!embed") {
        
        let embed = new Discord.MessageEmbed()
        .setTitle( "Toto se zobrazí nahoře na zprávě")
        .setDescription( "Řádek 1\nŘádek 2")

        return message.channel.send(embed);
    }

})

client.login(process.env.token);

Krok 5.

Půjdeš na discord devlopers portal.

V pravo nahoře klikni na New Application a tam si udělej novou aplikaci.

V levo klikni na Bot a tam klikni na Add Bot a pak na Yes, do it!

Potom klikni na Copy token

Krok 6.

Klikni v levo na ikonu zámku a do key napíšeme token.

Do value napiš token který sis právě kopíroval.

Krok 7.

Když se zase vrátíme.

Do value napiš token který sis právě kopíroval.

Krok 8.

Bota zapni pomocí tlačítka Run.

V consoli se ti zobrazí odkaz na bota který použij k pozvání bota k sobě na server :D
